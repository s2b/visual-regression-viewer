# Visual Regression Viewer

This is the complementary viewer application to inspect and edit report files
generated by [visual-regression-tester](https://github.com/s2b/visual-regression-tester).
To get started, please check out the README file of the main repository.

Note that this application (in contrast to other parts of this project) has mostly been
created with a coding agent. The code has been reviewed and finalized by a human.
